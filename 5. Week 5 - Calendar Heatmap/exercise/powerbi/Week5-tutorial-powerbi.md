# Week 5 - Visualization (PowerBI)

## Goal

Create a **Calendar Heatmap** (Matrix) of weekly distances.

## Instructions

### 1. Get Data

1.  Load `Week 5/data/weekly_distance_heatmap.csv`.

### 2. Create Visual

1.  Select **Matrix** visual.
2.  Drag `Year` to **Rows**.
3.  Drag `Week` to **Columns**.
4.  Drag `TotalDistance` to **Values**.

### 3. Format (Heatmap Effect)

1.  Select the Matrix.
2.  Go to **Format your visual > Cell elements**.
3.  Enable **Background color**.
4.  Click the **fx** icon next to Background color.
    *   **Format style**: Gradient.
    *   **Minimum**: White.
    *   **Maximum**: Purple.
    *   Click OK.
5.  **Grid**:
    *   Reduce padding to 0 for a tight "heatmap" look.
    *   Remove Alternating Row Colors if present.
