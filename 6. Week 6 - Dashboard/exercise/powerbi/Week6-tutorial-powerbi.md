# Week 6 - Visualization (PowerBI)

## Goal

Assemble a **Dashboard** (Report Page) combining visuals from Weeks 1-4.

## Instructions

### 1. Create New Page

1.  Click the **+** icon at the bottom to add a new page.
2.  Rename it **"Run Dashboard"**.
3.  **Page Settings**:
    *   Go to Format Page > Canvas Background.
    *   Set Color to Light Gray or White.

### 2. Copy Visuals

1.  Go to your **Week 1** page. Select the Bar Chart. **Ctrl+C**.
2.  Go to **Run Dashboard**. **Ctrl+V**.
3.  Repeat for:
    *   Scatter Plot (Week 2)
    *   Line Chart (Week 3)
    *   Donut Chart (Week 4)

### 3. Layout

1.  Arrange them in a 2x2 grid.
    *   Top Left: Monthly Bar Chart.
    *   Top Right: Scatter Plot.
    *   Bottom Left: Line Chart.
    *   Bottom Right: Donut Chart.
2.  **Align**: Use **Format > Align** tools to ensure they snap to a grid.
3.  **Titles**: Ensure all titles are consistent in font size (e.g., DIN, Size 12).
