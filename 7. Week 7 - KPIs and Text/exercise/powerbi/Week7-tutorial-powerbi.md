# Week 7 - Visualization (PowerBI)

## Goal

Create **KPI Cards** for high-level statistics.

## Instructions

### 1. Get Data

1.  Load `Week 7/data/kpi_stats.csv`.

### 2. Create Visuals

1.  Select **Card** visual (The one with "123" icon).
2.  Drag `TotalRuns` to the **Fields**.
    *   Label matches column name "TotalRuns".
3.  Create another Card for `TotalDistance`.
4.  Create another Card for `LongestRun`.

### 3. Format

1.  **Callout Value**:
    *   Font: Bold.
    *   Size: 30px.
    *   Color: Purple.
2.  **Category Label**:
    *   Color: Dark Grey.
3.  **Arrangement**:
    *   Place them in a row at the top of your Dashboard page (Week 6).

### 4. Smart Narrative (Optional)

1.  Insert **Smart Narrative** visual.
2.  Power BI will attempt to generate text summaries automatically.
