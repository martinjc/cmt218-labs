# Week 8 - Visualization (PowerBI)

## Goal

Apply **Conditional Formatting** to charts based on values.

## Instructions

### 1. Select Chart

1.  Select your **Monthly Bar Chart** (Week 1).

### 2. Configure Colors

1.  Go to **Format visual > Columns**.
2.  Locate **Color**. Instead of picking a single color, click the **fx** icon.

### 3. Set Rule

1.  **Format style**: Gradient.
2.  **What field should we base this on?**: `Sum of TotalDistance`.
3.  **Minimum**: Light Purple.
4.  **Maximum**: Dark Purple (or a contrasting color like Orange for high intensity).
5.  Click **OK**.

### 4. Result

The bars should now change color intensity based on their height (distance).
